# Two-stage Temporal Modelling Framework for Video-based Depression Recognition using Graph Representation

这个资源库包含一个基于研究的项目，通过AVEC 2019数据集检测一个人的抑郁程度。在这个项目中，通过MTA、NS和 SPG进行预测分析，三个module的详细说明请见paper。

## 使用说明
1. 代码clone 到本地机器。
2. 将AVEC 2019数据集进行下载。
